# LangGraph Agentic AI Project

## Description
A LangGraph-based agentic AI application with Streamlit UI.

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python app.py
```

## Project Structure
- `app.py` - Main application entry point
- `src/langgraphagenticai/` - Core application package
  - `graph/` - Graph building components
  - `LLMS/` - LLM integrations
  - `nodes/` - Graph node implementations
  - `state/` - State management
  - `tools/` - Agent tools
  - `ui/` - User interface components
